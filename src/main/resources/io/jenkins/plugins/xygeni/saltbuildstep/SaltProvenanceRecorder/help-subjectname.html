A collection of name and value to add to attestation. Value could be some of following:
  • value to add to attestation, typically a digest of the actual content like 'sha256:HEX_ENCODED_HASH'.
  • file to add to attestation, as a path relative to the workspace.
  • image to add to attestation, published container image, [REG/]NAME[:TAG]
    Example: index.docker.io/my_org/my_image:latest or my_org/my_image.
  • digest to add to attestation, prefixed by the digest function
    Example: sha256:d82938...8e62.
